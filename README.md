# 🚢 Batalha Naval - Java

## 📖 Descrição do Projeto

Este projeto consiste na implementação do jogo **Batalha Naval** utilizando a linguagem **Java**, desenvolvido como trabalho da disciplina de **Programação Orientada a Objetos (POO)**.

Diferentemente da versão tradicional entre dois jogadores, nesta implementação o usuário enfrenta uma **máquina (computador)**, que realiza ataques automaticamente seguindo a lógica definida no sistema.

O projeto foi desenvolvido com o objetivo de aplicar conceitos fundamentais da orientação a objetos, modelando os elementos do jogo por meio de classes, objetos e relacionamentos.

---

## 🎯 Objetivos

* Aplicar os princípios da Programação Orientada a Objetos.
* Desenvolver um jogo interativo baseado em turnos.
* Implementar a lógica de tomada de decisão da máquina.
* Trabalhar com matrizes, classes e estruturas de controle em Java.
* Praticar desenvolvimento colaborativo em equipe.

## 🎮 Funcionamento do Jogo

O jogo ocorre entre:

* 👤 **Jogador**
* 🤖 **Máquina (Computador)**

Cada participante possui um tabuleiro contendo seus navios. Durante a partida, o jogador escolhe coordenadas para atacar o tabuleiro adversário, enquanto a máquina realiza ataques automaticamente.

O jogo termina quando todos os navios de um dos participantes forem destruídos.

---

## 🎯 Regras do Jogo

1. Os navios são posicionados no início da partida.
2. O jogador e a máquina atacam alternadamente.
3. Cada ataque pode resultar em:

   * Água (erro)
   * Acerto
   * Navio afundado

4. O jogador não pode atacar a mesma posição duas vezes.
5. A máquina também evita repetir ataques já realizados.
6. Vence quem destruir todos os navios adversários primeiro.
